# 🧩 XMLEditor RM

**XMLEditor RM** é uma aplicação intuitiva para edição e validação de arquivos XML, com recursos visuais modernos, suporte a tema escuro, backup inteligente e substituição precisa de conteúdo.

## ✨ Funcionalidades

- 🔍 Busca de eventos XML com destaque e comparações
- 🧪 Validação de estrutura XML e destaques de sintaxe
- 🛠️ Substituição de texto: individual ou em lote
- 💾 Backup automático e visualizador embutido
- 🌓 Alternância de tema claro/escuro
- 🔁 Suporte a arquivos grandes e atalhos úteis

## 🚀 Como usar

1. Clone este repositório ou baixe o instalador
2. Execute `XMLEditor_Installer.exe` e siga as instruções
3. Comece a editar seus arquivos XML com estilo 🖤

## 💼 Requisitos

- Windows 10/11
- Python 3.10+ (caso use o `.py`)
- Módulos usados:
  - `tkinter`
  - `os`, `sys`
  - `pyinstaller` (para empacotar)

## 📦 Instalador

Para usuários finais, basta baixar o executável:
> [`XMLEditor_Installer.exe`](link-aqui-se-hospedar)
